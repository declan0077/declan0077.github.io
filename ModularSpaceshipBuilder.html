Game Overview

About the Game

"Modular Spaceship Project" is an evolved 2D space game where players design and pilot custom spacecraft using a modular component system. Engage in dynamic combat, manage resources, and navigate faction relationships in a procedurally generated universe.

Inspired by classics like Captain Forever, this reimagined version introduces advanced ship mechanics, faction dynamics, and resource management, built from the ground up with lessons learned from my previous attempt.

Game Details

  • Platform: PC (Steam)
  • Engine: Unity
  • Status: Coming Soon
  • Controls: Mouse (Drag/Connect), Space (Thrust), M (Mirror), Q/E (Rotate), Right Click (Collect Cargo), Left Click (Weapons), S (Shields)

System Requirements

Minimum Requirements

  • OS: Windows 10
  • Processor: Intel Core i3-2100 / AMD FX-6300
  • Memory: 4 GB RAM
  • Graphics: DirectX 11 compatible
  • DirectX: Version 11
  • Storage: 2 GB available space

Recommended Requirements

  • OS: Windows 10 or newer
  • Processor: Intel Core i5-4590 / AMD Ryzen 3 1200
  • Memory: 8 GB RAM
  • Graphics: GTX 1050 / RX 560 or equivalent
  • DirectX: Version 11
  • Storage: 4 GB available space
