Project Overview

Keeping with practicing, I developed two classic card game implementations: Blackjack and Hi-Lo. These projects were designed to explore various programming concepts and challenge myself with game logic and user interaction. Additionally, I experimented with doTween, a package I was aware of but hadn't previously used extensively.

Blackjack Game

The Blackjack game focused on implementing core casino mechanics, such as card dealing, scoring, and player/dealer decision-making. Developing the project turned out to be more interesting than anticipated, involving nuanced logic challenges.

  • Dealer Standing Logic: Ensuring the dealer stands when their hand value is 17 or higher.
  • Blackjack Detection: Checking for a natural blackjack at the start of each round.
  • Push and Bust Scenarios: Handling edge cases with ties or exceeding 21.

Hi-Lo Game

While simpler, the Hi-Lo game provided an engaging card prediction experience. This project allowed me to experiment with odds, predictions, and Unity text features. In hindsight, I should have developed this first, as it contains much of the same initial logic as Blackjack.
